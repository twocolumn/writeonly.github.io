



Kolejność parametrów w funkcji


Kolejność imperatywna

```java
  A functionName (B b, C c, D d);
```
 
 
```scala
   functionName(D d, C c, B b): A
```

```haskell
   functionName D -> C -> B -> A 
```

```haskell
   functionName d -> c -> b -> a 
```

```haskell
   functionName d -> (c -> (b -> a)) 
```


```haskell
   functionName b -> a -> a
```

```haskell
   functionName b -> a -> Bool
```
